## Getting Started

### Prerequisites
- Node.js (v14 or later)
- npm (v6 or later)

### Quick Start
```
# Clone the repo
git clone https://github.com/ignaciomercado4/noted.git

# Navigate to project
cd noted-master

# Install dependencies
npm install

# Create SQLite database
touch database.sqlite

# Start development server
npm run dev
```
